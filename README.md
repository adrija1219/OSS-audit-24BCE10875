## Scripts Overview

### Script 1 — System Identity Report (script1.sh)
Displays system information like a welcome screen including
distribution name, kernel version, logged in user, home directory,
date and time, and open source license information.

### Script 2 — FOSS Package Inspector (script2.sh)
Checks whether Python is installed on the system, displays
its version and location, and prints a one line open source
philosophy note using a case statement.

### Script 3 — Disk and Permission Auditor (script3.sh)
Loops through important system directories and reports
their size and permissions using a for loop. Also checks
the Python installation directory specifically.

### Script 4 — Log File Analyzer (script4.sh)
Reads a log file line by line, counts how many lines contain
a specific keyword, and prints the last 5 matching lines.
Creates a sample log file automatically if none is provided.

### Script 5 — Open Source Manifesto Generator (script5.sh)
Asks the user three questions interactively and generates
a personalised open source philosophy statement, saving
it to a text file.

## How To Run Scripts on Linux

### Dependencies Required
- Linux system (Ubuntu/Debian recommended)
- Bash shell
- Python3 installed

### Installation
Clone this repository on your Linux system:
\```bash
git clone https://github.com/adrija1219/oss-audit-24BCE10875
cd oss-audit-24BCE10875
\```

### Give Execute Permission to All Scripts
\```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
\```

### Run Each Script

**Script 1 — System Identity Report**
\```bash
bash script1.sh
\```

**Script 2 — FOSS Package Inspector**
\```bash
bash script2.sh
\```

**Script 3 — Disk and Permission Auditor**
\```bash
bash script3.sh
\```

**Script 4 — Log File Analyzer**
\```bash

\```

**Script 5 — Manifesto Generator**
\```bash

\```
